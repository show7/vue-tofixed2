# vue-tofixed2

> 由于js中进行小数计算会有精度问题，此方法用于金额计算（保留小数点后两位小数，并进行四舍五入）

## Build Setup

``` bash
# 安装
npm install vue-tofixed2

# import Rounding from 'Rounding'
在Main.js中引入

# Vue.use(Rounding)
全局注册Rounding

# this.operations(number1,number2,'运算符号')
组件中使用方法

例子 :计算 0.1 + 0.2
this.operations(0.1,0.2,'+')
